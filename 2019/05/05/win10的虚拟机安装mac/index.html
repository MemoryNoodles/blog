win10的虚拟机安装mac

发表于 2019-05-05 |

前言

为了react-native能在windows系统上也可以打ios包，于是就在windows系统上创建了虚拟机安装了mac。以下是我安装的过程。希望对你有帮助

工具准备

1、VMware 15（其他附近版本都没问题），下载地址：
链接: https://pan.baidu.com/s/1su3PckIuEc_bP1EPvvPaBA 提取码: svrp 复制这段内容后打开百度网盘手机App，操作更方便哦
永不过期序列号：UZ792-DHF8J-M81XP-MGM5T-MCAF2
2、下载“unlocker-masterv3.0.zip
链接: https://pan.baidu.com/s/1nzVIKGmwVVPBtNAduvpgdw 提取码: gxps 复制这段内容后打开百度网盘手机App，操作更方便哦
3、Mac OS 10.13 镜像
链接: https://pan.baidu.com/s/1eA0COEZpJUrt4d-tUSqpRQ 提取码: dxqc 复制这段内容后打开百度网盘手机App，操作更方便哦

安装步骤

1、首先找到VMware-workstation-full-15.0.0-10134415.exe文件双击运行；
2、解压unlocker-masterv3.0，进入文件夹以管理员的身份运行win-install.cmd文件；
3、打开 VMware ，创建新虚拟机，选择【典型】，后面的步骤具体参考了https://blog.csdn.net/qq_40147863/article/details/84797618这个前面的步骤讲的多一点和https://www.jianshu.com/p/4413fbe97e88 -》这个后面讲的多一点

系统全屏显示

安装好系统后，系统界面很小，要使其全屏显示还要安装VMware tools,这时候最好关闭系统，再从新开一次。点击虚拟机菜单下的“安装VMware tools”，完了之后重启就可以了
