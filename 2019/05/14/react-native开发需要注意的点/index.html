react-native开发需要注意的点

发表于 2019-05-14 |

前言

这一两个星期用react-native做了一个demo,其中遇到的一些问题在这里记录下来，方便以后查阅。如果你也遇到相同的问题，希望这些答案对你有用。

路由

我使用的是官网推荐的react-navigator，但是安装完了并没完这时还需要安装react-native-gesture-handler,并在命令行使用react-native link react-native-gesture-handler命令，这些步骤完了之后，还需要改安卓下的MainActivity.java这个文件

常规问题

（1） View：不能被添加事件，使用Touchable系列，一般TouchableWithoutFeedback。
（2）模拟器突然黑屏，APP启动不起来（闪退），可能是代码样式错误（margin, padding）导致的
（3）FlatList 在View、TouchableOpacity后面不能滚动
（4）想用空白标签来写样式，没用

UI组件

使用native-base, 原因是对比了好几个UI组件，发现这个组件库的组件最多，呵呵。自己属于比较懒的那种，外面有的好的东西直接拿过来就用了，如果没有或不好用再自己写。上次就自己为公司制定了一个脚手架
